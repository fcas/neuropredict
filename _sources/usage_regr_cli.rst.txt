

Regression workflow
--------------------------------

Once :doc:`installation` is successful, the usage for this workflow can be obtained by typing either of these following commands in the terminal.

.. code-block:: bash

    neuropredict_regress
    np_regress -h


Those options are also shown below:

.. argparse::
   :module: neuropredict.regress
   :func: get_parser_regress
   :prog: neuropredict_regress
   :nodefault:
   :nodefaultconst:
